Dos 二つ

Carole Bloch

Richard MacIntosh

Karina Vásquez

Áurea Vericat

スペイン語

レベル 1

Dos pequeñas manos para sostener.

握るための二つの小さな手

Dos pequeños pies para patear.

蹴るための二つの小さな足

Dos pequeños ojos para ver.

見るための二つの小さな目

Dos pequeños oídos para escuchar.

聞くための二つの小さな耳

¡Y dos brazos amorosos para ABRAZAR!

そして抱きしめるための二つの愛情に満ちた腕！
