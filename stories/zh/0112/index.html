我的身体 僕の体 My body

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini

dohliam

Zhuo Sun

中国語

レベル 1

让我能跑步。

僕は走れるよ。

I can run with it.

让我能跳跃。

私はジャンプができるわ。

I can jump with it.

让我能跳舞。

僕はダンスができるよ。

I can dance with it.

让我能游泳。

僕は泳げるよ。

I can swim with it.

让我能跳绳。

私は縄跳びができるわ。

I can skip with it.

让我能踢球。

僕は蹴ることができるよ。

I can kick with it.

让我能逃走。

危ないときには逃げられるよ。

I can escape with it.

可是总不能让我飞！

だけどどうしても空は飛べないんだ。

But I can never fly with it.
